# SEO Notes

## Production URL

The current canonical URL is:

```text
https://ranishrajshrestha.com/
```

Update this URL in these files if the deployed Vercel or custom domain is different:

- `index.html`
- `src/components/SEO.tsx`
- `public/robots.txt`
- `public/sitemap.xml`

## After Deployment

1. Add the production domain to Google Search Console.
2. Submit `https://ranishrajshrestha.com/sitemap.xml`.
3. Use URL Inspection to request indexing for the homepage.
4. Check PageSpeed Insights and fix major Core Web Vitals issues.
5. Add links to the portfolio from GitHub, LinkedIn, project READMEs, and relevant profiles.
